🏏 IPL Winning Percentage Predictor
An end-to-end Machine Learning web application that predicts the winning probability of the chasing team in an IPL match based on real-time match scenarios.

📝 Overview
Cricket is a game of uncertainties, but with data, we can find patterns. This project uses historical IPL data (2008-2024) to calculate the win probability of the second batting team. It considers factors like:

Target Score

Current Score

Overs Completed

Wickets Fallen

Venue & Teams

🚀 Features
Real-time Prediction: Enter current match stats to get instant win/loss percentages.

Dynamic UI: Interactive sliders and dropdowns for ease of use.

Probability Bars: Visual representation of how the match is tilting towards a team.

Accurate Model: Trained on thousands of IPL deliveries using Logistic Regression/Random Forest.

🛠️ Tech Stack
Language: Python 3.x

Libraries: Pandas, NumPy, Scikit-Learn

Framework: Streamlit (for Web UI)

Dataset: Kaggle (IPL Ball-by-Ball data)
